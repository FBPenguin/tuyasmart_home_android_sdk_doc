## User Data Model

**User**

| Parameters | Description                                  |
| ---------- | -------------------------------------------- |
| nickName   | Nickname                                     |
| phoneCode  | Country code                                 |
| mobile     | Mobile phone number                          |
| username   | User name                                    |
| email      | Email                                        |
| uid        | Unique user identifier                       |
| sid        | Login generates the unique identification id |
| headPic    | Url of the User Account Picture              |
